Техническое задание на доработку программы.
---

# 1. Пользователь

* Имя пользователя **создаётся при первом запуске**.
* Имя хранится в **LocalStorage**.
* Оно **не меняется между играми**.
* Все локальные игры принадлежат **этому одному пользователю**.

Пример:

```
localStorage
player_name = "Alexander"
player_guid = "8f3e1c..."
```

---

# 2. Локальная история игр

## Где хранится

LocalStorage.

## Что хранится

Каждая игра — отдельная запись.

Структура записи:

```
{
  score: 120,
  correctAnswers: 24,
  level: "Level 2",
  date: "2026-03-10T19:21:00Z"
}
```

## Сортировка

История всегда сортируется:

```
по дате
новая игра сверху
```

---

# 3. Локальная таблица (история игрока)

Показывается **только текущий пользователь**.

Колонки:

| Punkte | Richtige | Row     | Datum            |
| ------ | -------- | ------- | ---------------- |
| 120    | 24/30    | x2 - x9 | 10.03.2026 19:21 |

Особенности:

* имя пользователя **НЕ отображается**
* все игры сохраняются
* лимита записей **нет**
* новая игра появляется **сразу после завершения**

---

# 4. Глобальная таблица результатов

Хранится **на сервере (Bitbucket)**.

## Структура данных

Для каждого пользователя хранится **только лучший результат**.

Пример:

```
[
 {name:"Anna", score:320},
 {name:"Tom", score:300},
 {name:"Max", score:280}
]
```

---

# 5. Обновление глобальной таблицы

После каждой игры:

1. Проверяется пользователь.
2. Если его **нет** — добавляется.
3. Если есть:

   * новый score **больше** → обновить
   * меньше → игнорировать

---

# 6. Сортировка глобальной таблицы

Всегда:

```
по score
по убыванию
```

# 8. Отображение текущего пользователя

Текущий пользователь **показывается ВСЕГДА**, даже если он не в топ-10.

### Возможны 2 варианта:

---

## вариант 1 (если игрок в топ-10)

Он уже находится в таблице.

Его строка **подсвечивается**.

Пример:

```
6  Alexander  210   ← подсветка
```

---

## вариант 2 (если игрок не в топ-10)

Тогда таблица выглядит так:

| Platz | Name      | Punkte |
| ----- | --------- | ------ |
| 1     | Anna      | 320    |
| 2     | Tom       | 300    |
| ...   | ...       | ...    |
| 10    | Julia     | 150    |
| ...   | ...       | ...    |
| 37    | Alexander | 80     |

То есть:

* показывается **топ-10**
* затем **строка текущего пользователя**

Она подсвечивается.
