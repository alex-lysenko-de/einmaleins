Нужно выполнить рефакторинг приложения, с целью превратить приложение в реальный развивающий проект, который можно выложить на Github и другие хостинги приложений: Microsoft Store, Google Play
Основные цели рефакторинга:

* **PWA**
* **масштабирование режимов**
* **чистый игровой движок**
* **простоту поддержки**
* **будущие версии (multiplayer, AI, новые уровни)**

---

# 4. GameEngine (сердце игры)

GameEngine управляет всей логикой.

```
src/game/engine/GameEngine.js
```

Пример:

```javascript
export class GameEngine {

  constructor() {
    this.state = null
  }

  startLevel(level) {
    this.state = createGameState(level)
  }

  selectCard(card) {
    return resolveCardSelection(this.state, card)
  }

  nextTask() {
    return generateTask(this.state)
  }

}
```

UI **не знает** ничего о логике.

UI просто вызывает:

```
engine.selectCard()
```

    currentTask: null,
    visualMode: "rows"
  }

}

---

### useGame

```javascript
import { GameEngine } from "../game/engine/GameEngine"

const engine = new GameEngine()

export function useGame() {

  function start(level) {
    engine.startLevel(level)
  }

  function selectCard(card) {
    engine.selectCard(card)
  }

  return {
    start,
    selectCard
  }
}
```
